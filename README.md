# Dự Án UI - Giao Diện Người Dùng

## Mô Tả Dự Án

Đây là dự án giao diện người dùng (UI) cho hệ thống quản lý, bao gồm hai phần chính: giao diện dành cho Admin (quản trị viên) và giao diện dành cho User (người dùng). Dự án được xây dựng bằng HTML, CSS và JavaScript để tạo ra các trang web tĩnh phục vụ cho việc quản lý và phân tích kết quả.

## Cấu Trúc Thư Mục

```
UI/
├── README.md                 # Tài liệu hướng dẫn dự án
├── Admin/                    # Thư mục giao diện Admin
│   ├── history.html          # Trang lịch sử
│   ├── home.html             # Trang chủ Admin
│   ├── model.html            # Trang mô hình
│   ├── report.html           # Trang báo cáo
│   ├── result_analysis.html  # Trang phân tích kết quả
│   └── user_managerment.html # Trang quản lý người dùng
└── User/                     # Thư mục giao diện User
    ├── index.html            # Trang chủ User
    └── result_analys.html    # Trang phân tích kết quả User
```

## Tính Năng Chính

### Giao Diện Admin
- **Trang chủ**: Tổng quan về hệ thống
- **Quản lý người dùng**: Thêm, sửa, xóa và quản lý tài khoản người dùng
- **Mô hình**: Quản lý các mô hình trong hệ thống
- **Báo cáo**: Xem và tạo báo cáo
- **Phân tích kết quả**: Phân tích dữ liệu và kết quả
- **Lịch sử**: Theo dõi lịch sử hoạt động

### Giao Diện User
- **Trang chủ**: Giao diện chính cho người dùng
- **Phân tích kết quả**: Xem và phân tích kết quả cá nhân

## Công Nghệ Sử Dụng

- **HTML5**: Cấu trúc trang web
- **CSS3**: Styling và thiết kế giao diện
- **JavaScript**: Tương tác và chức năng động

## Cách Chạy Dự Án

1. **Clone hoặc tải xuống dự án**:
   ```
   git clone <repository-url>
   cd UI
   ```

2. **Mở các file HTML**:
   - Mở trực tiếp các file `.html` trong trình duyệt web
   - Hoặc sử dụng một máy chủ web local như Live Server trong VS Code

3. **Truy cập**:
   - Admin: Mở các file trong thư mục `Admin/`
   - User: Mở các file trong thư mục `User/`
